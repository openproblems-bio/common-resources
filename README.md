# common-resources
This repo contains common resources that can be used for OpenProblems v2 tasks.

## Submodule
To use this repo within your task you will need to add it as a submodule.

In your repo perform following git command:

```
git submodule add git@github.com:openproblems-bio/common-resources.git common
```

To update the submodule follw the below instructions:

* `cd common`
* `git checkout`
* `cd ..`
* commit the changes in the main repo

## Template

When starting a new task you can use the [task-template](https://github.com/openproblems-bio/task-template) repo where the `common-resources` repo has already been added as submodule.
